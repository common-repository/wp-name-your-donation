This plugin allows website users to donation any amount they'd like to.
== Description ==
Use a shortcode to embed a form anywhere on your website.  This form allows people to make donations and they can set any amount they'd like.

Currently supporting authorize.net, stripe and paypal with more to come.  Please feel free to suggest merchant gateways you would like to see.

An Email is sent to you and the user donating

== Installation ==
Simply upload the plugin to to your wp-content/plugins directory and activate.

1) It's as simple as using a shortcode and setting your gateway.  Go to the Settings > WP Name Your Donation page and grab the shortcode for the merchant gateway you'd like to use.  Be sure to set your API keys for the merchant
